# concept_map —— 加速器物理概念关系地图

> 用途:把概念间的依赖/因果关系画出来,支撑 reference_locator_policy §1

> 用法:遇到宽问题先在这里找到对应子树,把每个叶子作为一条独立检索去 retrieve.py 取证。

## 束损 (beam loss) 的成因展开

```
束损 beam loss
├── 单粒子动力学
│   ├── 动力学孔径不足 (dynamic aperture)
│   ├── 共振穿越 (resonance crossing)
│   ├── 闭合轨道畸变 (closed-orbit distortion)
│   └── RF 俘获损失 / RF 桶溢出
├── 束–气相互作用
│   ├── 弹性散射 (elastic / Coulomb)
│   └── 非弹性散射 (bremsstrahlung)
├── 束内 / 束团内
│   ├── Touschek 散射(大角,致损)
│   └── 束内散射 IBS(小角,致发射度增长)
└── 集体效应
    ├── 空间电荷致调谐扩散 → 共振 → 损失
    └── 各类不稳定性 (instabilities)
```

## 空间电荷 (space charge) 的效应链

```
空间电荷 self-field
├── 来源:束自身电荷/电流 → 平均场/自洽势(主导)
│        + 像电荷(真空室壁,间接)
├── 标度:∝ 流强;∝ 1/γ²(高能被磁自场抵消)
├── 横向后果
│   ├── 散焦力 → 抵消外部聚焦 → 束尺寸增大
│   ├── 非相干调谐移 Δν<0(Laslett) → 调谐扩散
│   │      └── 工作点扫过共振 → 发射度增长 / 束损
│   └── 相干模式频移(相干不稳定性)
├── 纵向后果:纵向空间电荷 → 微波不稳定性(过渡能量附近)
└── 制度:空间电荷主导 vs 发射度主导(看自场力 vs 发射度压力)
```

## 发射度增长 (emittance growth) 的机制

```
发射度增长 emittance growth
├── 失配 (mismatch) → 包络振荡 → 相空间稀释
├── 空间电荷非线性 → 共振、halo 形成
├── 束内散射 IBS(尤其强流/低能/冷束)
├── 散射(束气、Touschek 的小角部分)
├── 色散/色品未校正下的滤波
└── 不稳定性饱和后的相空间扰动
注:与之相反的是"冷却"——若文中出现"冷却同时发射度增大"须给机制,否则违背直觉。
```

## 集体不稳定性 (collective instability) 的分类

```
不稳定性 instability
├── 横向
│   ├── 头尾不稳定性 (head-tail)
│   ├── 耦合束团 (coupled-bunch)
│   └── 阻性壁 (resistive wall)
├── 纵向
│   ├── 微波不稳定性 (microwave)
│   └── 耦合束团(纵向)
├── 驱动:尾场/阻抗 (wakefield / impedance)、束流负载、电子云
└── 致稳:朗道阻尼、反馈系统、色品调节
```

## 纵向动力学 (longitudinal dynamics) 主线

```
纵向动力学
├── RF 腔 → 同步相位 φ_s → RF 桶
├── 同步振荡 (synchrotron oscillation)
├── 相滑移因子 η = α_p − 1/γ² → 过渡能量 γ_t
│      └── 过渡跃迁:相稳定性翻转、束团行为突变
├── 俘获 (capture):绝热俘获减少损失
└── 束团压缩 / 束团长度 ↔ 能散(纵向发射度守恒约束)
```

## 单粒子横向光学 (transverse optics) 主线

```
横向光学
├── 格子 lattice(二极/四极/六极) → 传输矩阵
├── Twiss (α, β, γ) → 束尺寸 √(βε) + 色散项 D·δ
├── 调谐 ν、相位推进 μ、工作点 (νₓ, ν_y)
├── 色品 ξ ← 六极校正
├── 共振条件 mνₓ+nν_y=p → 共振图、停止带
└── 动力学孔径(非线性长期稳定性)
```
